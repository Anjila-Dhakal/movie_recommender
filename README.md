# Movie Recommender

A Flask web application that recommends movies based on user preferences and ratings.

## Features
- User authentication
- Movie ratings
- Review system
- Movie recommendations using various algorithms
- Admin dashboard for managing users and movies

## Getting Started

### Prerequisites
- Python 3.6+
- Flask

### Installation
1. Clone this repository
2. Install the required packages using `pip install -r requirements.txt`
3. Run the application using `flask run`
